# diffyq

## installation

first clone the repo:

```bash
git clone https://github.com/elijw/diffyq
cd diffyq

```

---

## windows

### setup

```powershell
python3 -m venv venv
venv\Scripts\Activate.ps1
python3 -m pip install -r requirements.txt
```

### run

```powershell
python3 qt/euler_approx.py
```

---

## macOS / linux

### setup

```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
```

### run

```bash
python3 qt/euler_approx.py
```
